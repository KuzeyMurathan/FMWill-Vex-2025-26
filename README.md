# FMWill VEX V5 Robotics 2025-2026


Robot code for the VEX V5 Robotics Competition Push Back 2025-2026 season for FMWill.

## Overview

This repository contains the Python source code for our competition robot competing in the VEX V5 Robotics Push Back challenge during the 2025-2026 season.

## Project Structure

- **src/** - Main robot source code
- **.vscode/** - VS Code configuration and project settings

## Building and Deployment

This project uses the VEX Robotics SDK. Build and deploy using VEX's provided tools and IDE.
